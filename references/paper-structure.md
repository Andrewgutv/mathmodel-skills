# Paper Structure

## Default Heading Order
Use this order for teacher-style contest papers:

1. `摘要`
2. `一、问题的提出`
3. `二、问题的分析`
4. `三、模型假设`
5. `四、符号说明`
6. `五、建模与求解`
7. `六、模型的检验/敏感性分析`
8. `七、模型的评价与推广`
9. `参考文献`
10. `附录`

## Section Intent
- `摘要`: Write last. Cover problem, method, result, and contribution.
- `一、问题的提出`: Restate the contest task and delivery target.
- `二、问题的分析`: Split sub-problems and identify mathematical structure.
- `三、模型假设`: State explicit assumptions and why they are acceptable.
- `四、符号说明`: Define symbols, indices, and units before heavy derivation.
- `五、建模与求解`: Place the main derivation, algorithm, data handling, and results here.
- `六、模型的检验/敏感性分析`: Check robustness, error, or sensitivity where applicable.
- `七、模型的评价与推广`: Summarize strengths, limits, and transferability.
- `参考文献`: Keep formatting consistent with the target template.
- `附录`: Put long derivations, code excerpts, or extra tables here only when needed.

## Writing Rules
- Prefer equations, numbered steps, and quantitative statements over narrative filler.
- Keep each figure and table referenced in the surrounding text.
- Define symbols before the first equation that uses them.
